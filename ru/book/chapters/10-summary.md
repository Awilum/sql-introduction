# Что дальше?

Вы сделали первый шаг в изучении SQL и познакомились с основами работы с реляционными базами данных. Теперь вы умеете получать данные из таблиц, фильтровать и сортировать результаты, использовать встроенные функции и писать первые SQL-запросы.

Однако это лишь начало пути.

Чтобы уверенно работать с базами данных и применять SQL в реальных проектах, важно продолжать обучение и, самое главное, регулярно практиковаться.

Для этого была создана последовательная серия курсов, которая позволяет шаг за шагом освоить язык SQL — от базовых запросов до продвинутых методов анализа данных.

## SQL Введение

Если эта книга основана на одноимённом курсе, вы можете пройти его в интерактивном формате. Курс содержит практические задания, автоматическую проверку решений и дополнительные упражнения, которые помогут закрепить материал.

**курс:** https://stepik.org/a/290855

## SQL для начинающих

Следующий курс посвящён более глубокому изучению языка SQL. Вы познакомитесь с новыми конструкциями, научитесь строить более сложные запросы и закрепите знания на большом количестве практических задач, используя единую учебную базу данных.

**курс:** https://stepik.org/a/133728

## SQL. Оконные функции

После освоения базового SQL вы сможете перейти к одной из самых востребованных тем современного анализа данных — оконным функциям. Курс познакомит вас с ранжированием, накопительными вычислениями, функциями смещения и другими мощными инструментами обработки данных.

**курс:** https://stepik.org/a/282146

## Практический тренажёр по SQL

Этот курс практически полностью состоит из задач различной сложности. Вы будете работать с несколькими базами данных, решать реальные практические задания и закреплять навыки, максимально приближённые к тем, которые используются в профессиональной деятельности.

**курс:** https://stepik.org/a/279399

Какой бы путь вы ни выбрали дальше, помните: SQL — это язык, который невозможно освоить только чтением. Чем больше запросов вы напишете самостоятельно, тем увереннее будете чувствовать себя при работе с реальными данными.

Желаю вам успехов в дальнейшем обучении и новых профессиональных достижений!
