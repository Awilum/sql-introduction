# SQL Введение

Это русская версия учебной книги **«SQL Введение»**.

Данный раздел содержит полный исходный текст книги на русском языке, а также все необходимые файлы для сборки финального документа (PDF, EPUB).

## О содержимом

В этой папке находится:

- `book/chapters/` — главы книги (Markdown)
- `book/images/` — изображения, используемые в книге
- `book/metadata.yaml` — метаданные книги (название, автор, описание)
- `book/template.typ` — шаблон оформления (Pandoc / Typst)

## Сборка книги

Сборка выполняется из корня проекта:

```bash
./scripts/build.sh
```

После сборки результат будет сохранён в папке:

```
output/
```

После сборки доступны:

* sql-introduction.pdf
* sql-introduction.epub

Важно:

* Все файлы в output/ являются автоматически сгенерированными
* Их не нужно редактировать вручную
* Они не должны коммититься в репозиторий

### О версии

Это русская локализация и основная версия книги, предназначенная для изучения SQL с нуля.

> Эта папка содержит исходные данные книги, а не финальный результат.  
> Финальные файлы всегда генерируются через систему сборки.
